# Go Task Manager API

A RESTful Task Management API built with Golang, Gin, GORM, and MySQL.

This project demonstrates backend development concepts including authentication, middleware, RBAC authorization, and clean layered architecture.

---

## Features

- User Registration
- User Login
- Bearer Token Authentication
- Role-Based Access Control (RBAC)
- Task CRUD Operations
- Authentication Middleware
- MySQL Database Integration
- Clean Layered Architecture

---

## Tech Stack

- Golang
- Gin Web Framework
- GORM ORM
- MySQL
- Bearer Token Authentication
- RBAC Authorization
